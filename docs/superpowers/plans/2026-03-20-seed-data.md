# Seed Data Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** H2 인메모리 DB에 개발용 시드 데이터를 자동 로드하여 bootRun 후 즉시 API 테스트 가능한 환경을 제공한다.

**Architecture:** `data.sql`로 Workplace 기초 데이터를 앱 시작 시 자동 INSERT하고, `.http` 파일로 Worker 등록 API를 호출하여 도메인 로직(보험 자격 판정 + 데드라인 생성)이 자동 적용되도록 한다.

**Tech Stack:** Spring Boot 3.3.7, H2, JPA (create-drop), IntelliJ HTTP Client

**Spec:** `docs/superpowers/specs/2026-03-20-seed-data-design.md`

---

## File Structure

| Action | File | Responsibility |
|--------|------|----------------|
| Modify | `src/main/resources/application.properties` | `defer-datasource-initialization=true` 추가 |
| Create | `src/main/resources/data.sql` | Workplace 8건 INSERT |
| Create | `http/seed-workers.http` | Worker 50+건 POST 요청 |

### Task 1: application.properties 설정 추가

**Files:**

- [ ] **Step 1: 설정 추가**
